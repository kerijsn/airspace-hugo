To write out an [ISO 8601](https://en.wikipedia.org/wiki/ISO_8601) timestamp in the current language, you can use the `date_i18n` shortcode:

Writing

```
{{%/* date_i18n "2020-10-20" */%}}
```

will result in

```
{{% date_i18n "2020-10-20" %}}
```
